# Alex API

Fictional 2022 final-project fixture for NotZero.

## Run locally

1. Install the project dependencies.
2. Copy `.env.example` to `.env`.
3. Set `DATABASE_URL` and optionally change `PORT`.
4. Run the local health-check test.
5. Start the API process manually.

These instructions demonstrate a bounded manual environment and deployment process. They do not claim production deployment experience.
